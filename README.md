# UniQBench

A Cross-Platform Benchmarking Framework for Hybrid Quantum-Classical Machine Learning.

## Objectives
- Evaluate QML models across multiple platforms
- Measure noise portability (NPS score)
- Compare simulation vs real hardware performance

## Platforms
- Qiskit (IBM Quantum)
- Cirq (Google)
- Amazon Braket
- NVIDIA CUDA-Q

## Structure
- benchmarks/
- datasets/
- results/
- notebooks/
